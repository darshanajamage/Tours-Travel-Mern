# 🌍 Tours Travel MERN Application

## 📌 Project Overview

Tours Travel is a full-stack web application developed using the MERN stack. It allows users to explore tour packages, view destinations, and manage travel bookings.

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ✨ Features

- User authentication
- Tour packages listing
- Tour details
- Booking management
- Responsive user interface
- REST API integration

## 📂 Project Structure


Tours-travel-MERN

├── frontend
└── backend


## 🚀 How to Run

### Backend


cd backend
npm install
npm start


### Frontend


cd frontend
npm install
npm start
